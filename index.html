Web development

THE WEB PLAN

How we create

The beginning

The first step in creating an amazing website is by finding a colour pallet and set of type that will look great with your brand. If you already have an existing style that you go for will will re-asses to see if its still relevant at the current time.

Wireframes and demos

Once we know the basics we will start by putting together a wire-frame, and getting the first demos out. At this point we will then be communicating with you and making the necessary changes, so you are happy with what you see!

Going live

Once the final addition of the website as been created we will take the website online, and you will be ready for your customers to check it out!
